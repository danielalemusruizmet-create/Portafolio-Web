# Portafolio-Web

Portafolio web educativo enfocado en el uso crítico, ético y reflexivo de la inteligencia artificial, diseñado para estudiantes de licenciatura en física de la Universidad Distrital Francisco José de Caldas.

## Contenido

- `index.html`: Página principal con secciones de recursos, datos personales y multimedia.
- `assets/css/styles.css`: Estilos personalizados basados en Bootstrap.
- `assets/js/script.js`: Componente interactivo de gráficos.
- `assets/images/`: Ilustraciones SVG para el contenido.
- `assets/audio/reflexion-ai.wav`: Recurso de audio para reforzar el aprendizaje.

## Uso

Abrir `index.html` en un navegador moderno o servir el directorio con un servidor local.
